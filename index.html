Web Programing

Na Escola do Comércio de Lisboana disciplina de Tecnologia 1 no 3ºTIIGR, iremos aprender progamação Web 😅 e abordaremos os principios básicos associados ao desenvolvimento para a web, apenas do lado front end, lado visual. Também sabemos que existe um lado back-end, que é um lado do servidor, mas não iremos abordar esse lado.

Aprender-se-ão as seguintes tecnologias 🌐:

  • HTML
  • CSS
  • JavaScript

De forma a consolidar todos os conhecimentos, serão feitos alguns laboratórios.
